How this course is structured

The semester is divided up into seven periods (of varying length), each focusing on a specific area of the syllabus.

At the start of each period, you will be given a description with the learning goals (stated as questions) and a mini project (or a number of smaller exercises) that has to be implemented before the end of the period.

At least one period will be marked as non-compulsory. This will give you a chance to:

  • dig into areas not planned for the semester, but with topics of your interest
In order to replace a period with self-defined content, you must create your own description as outlined above. This must be approved before you start.
