What is JQuery

Query is a fast, small, and feature-rich JavaScript library.
It makes things like HTML document traversal and manipulation, event handling, animation, and Ajax much simpler with an easy-to-use API that works across a multitude of browsers.
With a combination of versatility and extensibility, jQuery has changed the way that millions of people write JavaScript.

Include JQuery on our pages

Two strategies:

1) Download JQuery from :http://jquery.com/download/ and include the file in JQuery pages like:

2) Include it from a CDN (Content Delivery Network

JQuery - The ready() function

The jQuery Function

The jQuery library defines a single global function named jQuery().

This function also defines the global symbol $ as a shortcut for it. These are the only two symbols jQuery defines in the global namespace.

Examples

$("div");   // selects all HTML div elements
$("#myId");// selects one HTML element with ID "myId"
$(".myClass"); // selects HTML elements with class "myClass"
$("ul, li, a.navigation");
// selects anchors with class "navigation" that are nested in list items
The value returned by these examples represents a set of zero or more DOM elements and is known as a jQuery objects

jQuery Event Handling

Hook on to the click event

$("p").on("click", function(){
  alert( $(this).text() );
});

Attach an event that runs only once

$("p").one("click", function(){
  alert( $(this).text() );
});

Remove all event handlers from all paragraphs

$("p").off()

In the context of jQuery this' is a DOM element when you are inside of a callback function being called for example by the click, each, bind, etc. methods.
To access basic javascript methods we can just use this, but to use jQuery functions you must use $(this)
